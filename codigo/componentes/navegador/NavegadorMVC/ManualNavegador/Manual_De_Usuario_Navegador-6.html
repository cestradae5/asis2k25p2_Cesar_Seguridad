Generalidades del sistema

 

 

Barra de navegación 

 

 

a)

 

Acciones CRUD (cuadro rojo)

 

a.

 

Botón ingresar: Habilita el resto de los botones cuando el usuario 
desea ingresar un registro o bien cualquiera de las operaciones 
básicas de un CRUD (Crear, Leer, Actualizar y Eliminar). De la misma 
forma crea los campos para ingresar los datos según la base de 
datos.

 

b.

 

Botón Modificar: Permite editar la información del registro que el 
usuario seleccione.

 

c.

 

Botón guardar: guarda los datos ingresados por el usuario o bien el 
registro que se haya modificado

 

d.

 

Botón cancelar: Deshabilita todos los botones, cancelando así 
cualquiera de las acciones que el usuario quería ejecutar o hacer.

 

b)

 

Referencia a otros componentes (cuadro verde)

 

a.

 

Botón consultar: abre el componente de consultas inteligentes

 

b.

 

Botón imprimir: abre el apartado de reporteador.

 

c)

 

Botón refrescar (flecha roja) : carga de nuevo el navegador para que se 
muestren los cambios realizados

 

d)

 

Botones de navegación (cuadro morado):

 

a.

 

Botón inicio: Lleva al usuario hacia el primer registro del la base
