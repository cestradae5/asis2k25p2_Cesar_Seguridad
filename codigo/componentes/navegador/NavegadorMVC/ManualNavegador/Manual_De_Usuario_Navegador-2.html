Introducción 

 

El presente manual de usuario tiene como finalidad guiar al lector en el uso del 
componente Navegador (DLL), una herramienta diseñada para facilitar la gestión 
de  información  dentro  de  los  módulos  del  sistema.  Este  componente  brinda  al 
usuario la posibilidad de desplazarse entre los registros disponibles, así como realizar 
de forma sencilla las operaciones básicas de crear, consultar, actualizar y eliminar 
(CRUD).

 

Además  de  las  funciones  de  navegación,  el  Navegador  se  integra  con  otros 
módulos  del  sistema,  como  las  Consultas  Inteligentes,  Reporteador  y  Seguridad, 
permitiendo  ampliar  las  opciones  de  consulta  y  generación  de  información.  De 
igual manera, el componente genera automáticamente las etiquetas y campos 
necesarios para la captura de datos, adaptándose a la tabla y a las columnas que 
correspondan en cada caso.

 

Este manual está orientado a los usuarios que utilizarán el Navegador en su entorno 
de trabajo diario. A lo largo del documento se presentan las funciones principales, 
su forma de uso y las recomendaciones necesarias para aprovechar al máximo sus 
beneficios.
